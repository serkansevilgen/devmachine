#Developer Machine Installer#

A simple script to install several tools to start coding on a freshly installed Ubuntu 12.04

This script will install packages below
```
Screen  
Htop  
Nethogs  
Curl  
Git  
ab  
pip  
virtualenv  
virtualenvwrapper  
fabric  
Nginx  
Node.js  
jshint  
forever  
express  
R  
RStudio Server  
Emacs 24  
MongoDB  
Heroku Toolbelt  
AWS Cli
Oracle Java 8  
```
